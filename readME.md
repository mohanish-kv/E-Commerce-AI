# AI Shopping Assistant

A Streamlit-based e-commerce demo that loads product and review CSV files, supports product search and filters, summarizes customer reviews, recommends similar products with TF-IDF cosine similarity, and includes a lightweight shopping chatbot.

## Features

- Product catalog loaded from `products.csv`
- Review data loaded from `reviews.csv`
- Search by name, brand, category, and description
- Category and price filters
- Product cards with ratings and reviews
- Rule-based review summaries with pros, cons, and verdict
- Similar-product recommendations using TF-IDF and cosine similarity
- Chat-style product recommendation assistant

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Structure

```text
app.py
products.csv
reviews.csv
requirements.txt
readME.md
```
